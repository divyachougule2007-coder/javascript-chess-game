# ♟️ JavaScript Chess Game

A simple and interactive **Chess Game** built using **HTML, CSS, and JavaScript**.  
This project allows users to play chess directly in the browser with a clean UI and basic game functionality.

---

## 🚀 Features

- Interactive chess board
- Piece movement using JavaScript
- Responsive and simple UI
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly project structure
